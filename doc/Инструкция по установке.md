## ЧАСТЬ 1. УСТАНОВКА ARDUINO IDE

### 1.1 Что такое Arduino IDE

**Arduino IDE** (Integrated Development Environment) — это бесплатная программа, в которой вы пишете код, компилируете его и загружаете на плату RP2040-ETH.

**Операционные системы:** Windows, macOS, Linux.

### 1.2 Скачивание установщика

1. Откройте браузер и перейдите на официальный сайт:  
   **https://www.arduino.cc/en/software**

2. На странице найдите блок **«Download the Arduino IDE»**

3. Выберите версию для вашей операционной системы:
   - **Windows:** нажмите `Windows Win 10 and newer`
   - **macOS:** нажмите `macOS Intel` или `macOS Apple Silicon` (в зависимости от процессора)
   - **Linux:** выберите соответствующий пакет (deb, rpm или AppImage)

> ** Рекомендация:** скачивайте версию **2.x** (новый интерфейс) или **1.8.x** (классический) — обе работают с RP2040.

### 1.3 Установка на Windows

1. Запустите скачанный файл (например, `arduino-ide_2.x.x_Windows_64bit.exe`)

2. Нажмите **«I Agree»** (Соглашаюсь) с лицензией

3. Нажмите **«Install»** (Установить) — можно оставить все настройки по умолчанию

4. Дождитесь завершения установки, нажмите **«Close»** (Закрыть)

> ** Важно:** если появляется окно «Контроль учётных записей» (UAC), нажмите **«Да»**

## ЧАСТЬ 2. ДОБАВЛЕНИЕ ПОДДЕРЖКИ ПЛАТЫ RP2040 В ARDUINO IDE (WINDOWS)

### 2.1 Что нужно сделать

После установки Arduino IDE необходимо добавить в неё информацию о плате RP2040. Без этого шага плата RP2040-ETH не определится.

### 2.2 Добавление ссылки на репозиторий плат

1. Запустите Arduino IDE

2. Откройте окно настроек: `Файл` → `Preferences` (Настройки)

3. Найдите поле **«Additional boards manager URLs»** (Дополнительные ссылки для менеджера плат)

5. Нажмите **«OK»**

### 2.3 Установка ядра поддержки RP2040

1. Откройте **Менеджер плат**: `Инструменты` → `Плата` → `Менеджер плат...`

2. В строке поиска введите **`rp2040`**

3. Найдите пакет **«Raspberry Pi Pico/RP2040»** от автора **Earle F. Philhower, III**

4. Нажмите кнопку **«Установить»**

5. Дождитесь завершения установки (это может занять несколько минут)

### 2.4 Выбор платы

1. Подключите плату **RP2040-ETH** к компьютеру

2. В меню `Инструменты` → `Плата` найдите раздел **«Raspberry Pi RP2040 Boards»**

3. Выберите **«Raspberry Pi Pico»**

### 2.5 Настройка порта

1. Убедитесь, что плата подключена к компьютеру

2. В меню `Инструменты` → `Порт` выберите появившийся COM-порт (например, `COM3`, `COM4` и т.д.)

> **Примечание:** Если порт не появился, возможно, потребуется установить драйвер для чипа CH340 или CP2102 (в зависимости от версии вашей платы).


5. Нажмите **«OK»**
